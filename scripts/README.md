# Scripts de Deploy e Utilidades

Este diretório contém scripts úteis para deploy e manutenção.

## Scripts Disponíveis

- `setup-vps.sh` - Configuração inicial da VPS
- `deploy.sh` - Deploy manual
- `backup.sh` - Backup do MongoDB
- `health-check.sh` - Verificação de saúde

## Uso

```bash
# Dar permissão de execução
chmod +x scripts/*.sh

# Executar script
./scripts/setup-vps.sh
```
